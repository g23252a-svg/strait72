# Strait 72 v0.3.1 UX Patch

사용자 플레이 테스트 피드백 반영 패치.

## 반영 사항

1. 총 턴 수 확대
   - 20턴 × 6시간 = 5일 → 30턴 × 6시간 = 7.5일
   - 72시간은 여전히 중국의 초기 속전속결 목표 시한으로 유지

2. AI 추천 반복 완화
   - UI 단에서 최근 추천 주공축/방어중점/카드에 감점 적용
   - 같은 추천만 반복되는 느낌을 완화
   - AI 추천 적용 시 주공축에 맞는 선택 지역도 자동 동기화

3. 새 게임 버튼 사고 방지
   - 새 게임 버튼을 턴 실행 버튼 옆에서 헤더 우측으로 이동
   - 클릭 시 확인창 추가

4. 표시 로직 정리
   - TURN/시간 표시의 20턴 하드코딩 제거
   - GAME_RULES.totalTurns 기준으로 표시

## 주의

이 패치는 밸런스 확정 패치가 아니라 UX/플레이 감각 개선 패치입니다. 30턴 구조로 인해 승률 분포는 다시 측정해야 합니다.
