# Strait 72 v0.3 Playable UI MVP

## 목적

이 패치는 기존 v0.2 엔진을 수정하지 않고, 브라우저에서 직접 턴을 눌러볼 수 있는 최소 HTML/Canvas 플레이 화면을 추가합니다.

## 추가 파일

```text
index.html
src/ui_canvas.js
src/playable_app.js
docs/ui_mvp_readme.md
```

## 실행 방법

프로젝트 루트에서 정적 서버를 켭니다.

```bash
python -m http.server 5173
```

브라우저에서 접속합니다.

```text
http://localhost:5173/
```

## 현재 가능한 것

- 중국 주공축 선택
- 대만 방어 중점 선택
- 선택 지역 지정
- 중국/대만 카드 체크 후 턴 실행
- AI 추천 적용
- 자동 1턴 진행
- 중앙 Canvas 지도 표시
- 좌우 자원 패널 표시
- 하단 미국/일본/한국/국제여론 게이지 표시
- 작전 로그 확인
- 엔진의 `runTurn()`으로 실제 상태 갱신

## 아직 의도적으로 안 넣은 것

- 손패/덱 드로우 시스템
- 블러프 카드 완전 해석
- 카드 사용 후 폐기/쿨다운
- 고퀄 맵 아트
- 사운드
- 승리/패배 전용 컷신
